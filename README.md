# Product Microservice

## Dev

### Clone the repository

### Installation

```bash
$ npm install
```

### Create File `.env` based in the `.env.template`

### Running migration prisma `npx prisma migrate dev`

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
